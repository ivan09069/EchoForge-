# Contributing to EchoForge

Thanks for your interest!

## How to Contribute
- Search existing Issues and Discussions before opening new ones.
- Use templates: "Bug report" or "Feature request".
- For PRs: small, focused changes with clear descriptions.

## Workflow
1. Fork and create a feature branch.
2. Add tests or verification steps when applicable.
3. Keep commits concise; reference Issues (e.g., "Fixes #123").
4. Open a PR; fill out the PR template; respond to feedback.

## Good First Issues
Look for `good first issue` and `help wanted` labels.
