# EchoForge

EchoForge is a zero-leakage, security-first portfolio tracker with FIDO2 biometric authentication, hardware-bound encryption, and real-time financial data feeds. Designed for privacy-conscious investors and professionals.

![Architecture Diagram](architecture/portfolio-tracker-architecture.svg)

## Key Features
- **FIDO2/WebAuthn Biometric Unlock** (Cross-device support)
- **Encrypted IndexedDB Storage** (AES-GCM, Hardware PRF key)
- **Real-Time Data Feeds** (CoinAPI, Polygon)
- **Fully Offline Capable**
- **Open Source, MIT Licensed**
